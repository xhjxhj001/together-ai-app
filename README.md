# together-ai-app

# 欢聚AI原生AI应用

# 简介
通过 Gradio 搭建 AI 原生应用前端页面，通过多模态大模型的能力，制作多种 AI引用。

## 一、快速使用
1. 将根目录下 lib/.env.template 复制一份到 lib/.env
`cp lib/.env.template lib/.env`
2. 配置 lib/.env 文件中的 SK 信息
3. 执行根目录下的 run.sh
`sh run.sh`

## 二、应用目录
1. AI 对话助手
2. 绘图助手
3. 儿童绘本生成（包含文字、图片、语音）
4. to be continued...
